# 📁 QRNG Post-Processing & Research Pipelines Hub

This directory contains your core research and development scripts for post-processing raw data logged from physical **Quantum Random Number Generators (QRNG)**. It includes privacy amplification pipelines, debiasing engines, serial hardware loggers, and timetag formatters.

---

## 📂 Sub-Folders & Modules

### 1️⃣ [`privacy_amplification_versions/`](file:///d:/Desktop/Codes/2_QRNG_Post_Processing_and_Pipelines/privacy_amplification_versions)
Contains successive evolutionary versions of your **Entropy Extraction & Privacy Amplification** pipeline. These scripts process binary bitstreams using hashing and mathematical lemmas to guarantee perfect entropy:
*   **`v1_base/`**: Basic Von Neumann debiasing algorithms and hash combinations.
*   **`v2_entropy_amp/`**: Adds an entropy amplification step before hashing.
*   **`v3_entropy_amp_optimized/`**: Optimizes the Von Neumann and LFSR loops for larger datasets.
*   **`v4_fast_leftover/`**: Implements a high-performance **Leftover Hash Lemma (LHL)** extractor using Toeplitz matrix hashing.
*   **`v5_unicode_fix_latest/`**: Resolves encoding errors when handling binary strings under modern Python runtimes. **(This is your primary, production-ready module!)**

### 2️⃣ [`paper_ready_pipelines/`](file:///d:/Desktop/Codes/2_QRNG_Post_Processing_and_Pipelines/paper_ready_pipelines)
*   **Description:** Consolidates your core academic evaluation pipelines (RCI Research & Quantum29 entropy duality).
*   **Key Files:** `paper_ready_qrng_pipeline_1.py`, `paper_ready_qrng_pipeline_2.py`, `QM20.py`, `QM20_entropy.py`.
*   **Features:** End-to-end processing that reads timetag binary records, calculates entropy bounds, executes privacy amplification extractors, and prepares paper-ready statistical graphs.

### 3️⃣ [`serial_logger/`](file:///d:/Desktop/Codes/2_QRNG_Post_Processing_and_Pipelines/serial_logger)
*   **Description:** Hardware interfacing scripts for logging data from serial hardware interfaces.
*   **Key Files:** `serial_reader.py`, `logger.py`, `datalogger.py`, `DRDO.png`, `DIAT.png`.
*   **Features:** Establishes multi-threaded RS232/USB serial communication to poll, decode, and log live bitstream sequences into compressed files for evaluation.

### 4️⃣ [`beacon_and_timetags/`](file:///d:/Desktop/Codes/2_QRNG_Post_Processing_and_Pipelines/beacon_and_timetags)
*   **Description:** Data formatters and parse utilities.
*   **Key Files:** `PYNQ_TT.py`, `Timetaggs_to_QRNGbits.py`, `becaon_genration.py`.
*   **Features:** Parses complex timetags generated by time-to-digital converters (like PYNQ boards) into raw, debiased bit representations suitable for extractor pipelines.
